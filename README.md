# sluiiice
A waterfall for iii grids, inspired by dam removal

- turn your grid so that the usb port faces upward

- the top row constantly pours fluid in but starts out blocked

- to start, tap a button in the top row to release the water

- press grid keys to add walls, the fluid will find its way down and around the walls, pooling where it can

- the bottom row drains fluid out. as fluid drains, it plays notes - the more fluid, the louder the note

-  adjust the musical variables at the top of the script to change its harmonic behaviour

- adjust the simulation variables to change the behaviour of the fluid simulation

- use update_scale(new_root, new_scale) in diii to change the scale and root note: e.g. update_scale("C#3", "minor")

- use update_bpm(new_bpm) in diii to change the bpm: e.g. update_bpm(240)
